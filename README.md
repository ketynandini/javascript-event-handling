JAVASCRIPT EVENT HANDLING
This project demonstrates how to handle user interactions in a web application using JavaScript event listeners and DOM manipulation.
Event covered:
- `click` — button click counter
- `keyup` — displays last key pressed
- `mouseover` / `mouseout` — hover zone effects
- `input` / `change` — live text preview & dropdown
- `focus` / `blur` — input field highlight
- `submit` — form validation without page reload
